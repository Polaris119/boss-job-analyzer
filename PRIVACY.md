# BOSS直聘岗位分析助手隐私政策

**生效日期：2026 年 7 月 14 日**

**最近更新：2026 年 7 月 14 日**

本隐私政策适用于 Chrome 扩展“BOSS直聘岗位分析助手”（以下简称“本扩展”）。本扩展由 GitHub 用户 [Polaris119](https://github.com/Polaris119) 维护，用于在用户主动选择岗位后，将岗位描述与用户提供的基础简历进行对比，生成岗位分析、准备建议、学习路线、面试题及可选的定制简历。

本扩展不是 BOSS直聘官方产品，与 BOSS直聘及其关联公司不存在隶属、授权或背书关系。

## 1. 单一用途

本扩展处理数据的唯一目的，是向用户提供其主动请求的岗位与简历分析功能，以及与该功能直接相关的任务管理、结果保存和导出能力。

本扩展不会将用户数据用于广告、用户画像、信用评估、出售数据、与核心功能无关的分析，或训练本项目自有模型。

## 2. 本扩展处理的数据

### 2.1 用户主动提供的简历数据

当用户上传 PDF 简历时，本扩展会在浏览器本机读取并解析其中的文字。简历可能包含用户主动提供的姓名、联系方式、教育背景、工作经历、项目经历、技能及其他个人信息。

本扩展目前仅支持文本型 PDF，不会将原始 PDF 文件上传至本项目开发者的服务器。

### 2.2 用户主动选择的岗位页面内容

当用户点击“AI 分析当前岗位”时，本扩展会读取当前 BOSS直聘岗位详情页中与分析有关的内容，例如：

- 岗位名称、公司名称、薪资及岗位标签
- 岗位描述和任职要求
- 当前岗位页面地址及用于识别岗位版本的标识

本扩展不会持续收集用户的浏览历史，也不会在用户未发起岗位分析时批量读取其他岗位页面。

### 2.3 AI 接口配置与认证信息

用户可以配置 AI 服务商、接口地址、模型名称和 API Key。

- 选择“记住 API Key”时，API Key 保存在 `chrome.storage.local`。
- 未选择“记住 API Key”时，API Key 保存在 `chrome.storage.session`，供当前浏览器会话使用。
- API Key 仅用于向用户配置的 AI 接口发送请求，不会发送给本项目开发者。

### 2.4 分析结果与任务数据

本扩展会在浏览器本机保存岗位任务及其执行状态、岗位与简历快照、岗位画像、差距分析、准备规划、面试题和可选的定制简历，以便用户查看结果、重试、重新分析、导出或删除。

## 3. 数据的使用方式

本扩展仅将上述数据用于：

- 解析并校对用户上传的基础简历
- 识别用户主动选择的岗位及岗位版本
- 生成岗位画像、简历差距、行动建议和学习路线
- 在符合基本投递条件时生成针对性面试题
- 根据用户选择生成定制简历
- 执行后台队列、失败重试、历史记录和结果导出
- 保存用户主动设置的扩展偏好

## 4. 数据存储

本项目不提供用于接收或存储用户简历、岗位内容、API Key 或分析结果的开发者服务器。

- 基础简历、AI 配置和扩展偏好保存在 Chrome 扩展本地存储中。
- 临时 API Key 保存在 Chrome 扩展会话存储中。
- 岗位任务、简历快照和分析结果保存在浏览器本机 IndexedDB 中。
- PDF 文本解析在浏览器本机完成。

这些本地数据不会通过本扩展进行跨设备同步。

## 5. 数据传输与第三方共享

用户发起分析后，本扩展会把完成该功能所必需的内容直接发送至用户选择和配置的 AI 服务商，包括岗位内容、基础简历快照，以及后续分析节点需要使用的中间结果。API Key 会通过请求认证头发送至该 AI 服务商。

除用户选择的 AI 服务商外，本扩展不会主动将用户数据共享给其他第三方，也不会将用户数据发送给本项目开发者。

不同 AI 服务商可能有各自的数据保留、日志记录和模型训练政策，这些行为由相应服务商负责，不受本项目控制。用户在配置和使用 AI 接口前，应阅读并接受所选服务商的服务条款与隐私政策。

除用户明确配置的本地开发地址外，本扩展要求 AI 接口使用 HTTPS 传输。

## 6. 数据保留与删除

- 基础简历会保留至用户在扩展内删除、清除扩展数据或卸载扩展。
- AI 配置会保留至用户修改、清除扩展数据或卸载扩展；会话 API Key 在对应浏览器会话结束后失效。
- 岗位历史默认保留最近 20 条，用户可以调整为 10、20 或 30 条。
- 超过历史上限时，本扩展会删除最早的已完成、失败或取消任务，以及其中保存的分析结果和简历快照。
- 等待中和运行中的任务不会因历史上限而被自动删除。
- 用户可以删除单条或批量历史任务，也可以清空历史记录。

卸载本扩展或通过 Chrome 清除本扩展的数据，会删除本机保存的数据。已经发送至 AI 服务商的数据如何删除，需要依据对应服务商提供的机制处理。

## 7. 权限说明

本扩展使用以下 Chrome 权限：

- `storage`：保存简历、AI 配置、偏好和任务状态。
- `sidePanel`：显示扩展首页与配置界面。
- `alarms`：在浏览器允许的情况下恢复和调度后台任务。
- `https://www.zhipin.com/*`：在 BOSS直聘页面展示分析入口，并在用户主动触发后读取当前岗位。
- 可选的 AI 接口域名权限：仅在用户保存对应接口配置时申请，用于向用户指定的 AI 服务商发送请求。

本扩展不读取 Cookie，不调用 BOSS直聘未公开接口，不自动投递简历，也不自动发送消息。

## 8. Chrome Web Store 数据使用承诺

本扩展对用户数据的使用遵守 Chrome Web Store User Data Policy，包括 Limited Use 要求。用户数据仅用于提供或改进本扩展明确说明的单一用途，不用于个性化广告、再营销、与功能无关的数据交易，也不会由开发者人工读取；法律要求、安全调查或用户主动提供特定数据以请求支持的情形除外。

## 9. 安全措施

本扩展通过以下方式降低数据风险：

- 使用最小化、按需申请的浏览器权限
- 默认要求 AI 接口使用 HTTPS
- 不设置用于收集用户数据的开发者后端
- 不嵌入广告、分析统计或行为追踪 SDK
- 将岗位读取和 AI 分析限定为用户主动触发的操作

任何本地软件都无法保证绝对安全。用户应妥善保管 API Key，并避免在公开 Issue、截图或日志中提交真实简历、API Key 或其他敏感信息。

## 10. 政策更新

如果本扩展的数据处理方式发生实质变化，本政策会同步更新生效日期和相关内容。需要新增数据用途或第三方共享方式时，会在功能启用前提供醒目说明，并在适用情况下取得用户明确同意。
