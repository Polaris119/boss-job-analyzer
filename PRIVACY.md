# 隐私说明草案

BOSS直聘岗位分析助手仅为生成用户主动请求的岗位分析和定制简历而处理数据。

## 处理的数据

- 用户主动上传并在本机解析的 PDF 简历内容
- 用户主动分析的当前职位页面内容
- 用户配置的 AI 接口地址、模型名称和 API Key
- AI 返回的岗位分析、学习路线和定制简历

## 数据用途

上述数据只用于插件的单一目的：分析用户选择的职位，并生成定制简历和学习建议。数据不用于广告、画像、出售或训练本项目自己的模型。

## 数据存储与传输

- 本项目不提供服务器；PDF 文件在本机解析，插件将配置保存在 Chrome 扩展存储中，将岗位任务、简历快照和分析结果保存在本机 IndexedDB 中。
- 用户点击分析后，职位和简历内容会直接发送给用户配置的 AI 服务商。
- API Key 仅发送到用户配置的 AI 接口，不会发送给本项目开发者。
- 用户应阅读并接受所选 AI 服务商的隐私政策。

## 用户控制

用户可以在插件内删除单项岗位记录、删除基础简历，或清空已完成、失败和取消的历史任务；等待中和运行中的任务会保留。用户也可以在 Chrome 中移除扩展或清除扩展数据，以删除包括 AI 配置在内的全部本地数据。

本文件是 MVP 阶段草案，Chrome Web Store 上架前需要补充开发者主体、联系方式、生效日期和公开访问地址。
